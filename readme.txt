== Description ==

**myCred LearnDash Points Importer** enables users to import points from LearnDash to myCred. These imported points are added to the myCred point balance and can be redeemed for different products. 

After synchronization between myCred Point Balance and LearnDash, the user can choose to override the LearnDash points system for myCred.

The plugin also has the ability to auto-fetch & display any default or custom point types created via myCred. 

**What is myCred?**
[myCred](https://wordpress.org/plugins/mycred/) is an intelligent and adaptive points management system that allows you to build and manage a broad range of digital rewards, including points, ranks, and badges on your WordPress/WooCommerce powered website.

**What is LearnDash?**
[LearnDash](https://www.learndash.com/) is a WordPress learning management system (LMS) trusted by companies, universities, training organizations, and entrepreneurs worldwide for creating (and selling) their online courses.

== Features ==
* Connect LearnDash with myCred.
* Synchronize LearnDash points with myCred Point Balance.
* Import points from LearnDash and add them to myCred Point Balance.
* Update point type logs with detailed entries.
* Notification after deduction of points from point balance.
* Override the LearnDash points for myCred Point Balance.
* Automatically fetch & display any default or custom point types created via myCred.

== Requirements ==

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
